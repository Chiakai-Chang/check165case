# 審核165案件小幫手

小小程式幫忙檢查受理時哪裡漏填或填錯<br>
還有一些溫馨小提醒<br>
自己就可以檢查，不用下班再接到電話說要修改囉！(讚讚讚)<br>

## 使用方法：<br>

* 1、在「165反詐騙平臺」開案受理完詐欺案件以後，請將反詐騙紀錄表PDF檔下載下來。<br>
* 2、將本程式打開，它會請您幫忙找到你所下載的PDF檔在哪。<br>
* 3、找到以後，它就會告訴您檢查結果啦！<br>
* 4、最後還可以將受理情形，透過本程式線上提交給所長、詐欺業務承辦人後續人工審查喔！<br>
